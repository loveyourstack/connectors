# TEDB

TEDB is the Taxes in Europe database. See https://ec.europa.eu/taxation_customs/tedb/#/home.

CN and CPA are European product and service classification codes:
* CN (Combined Nomenclature): a coding system used for the classification of -goods-: https://data.europa.eu/data/datasets/combined-nomenclature-2025?locale=en
* CPA (Classification of Products by Activity): a classification system for -services-: https://ec.europa.eu/eurostat/web/cpa
