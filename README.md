# connectors - API client code, database stores and sync functions

Packages for accessing and saving API data.

Only available for PostgreSQL.

## Available connectors

### AWS
`Requires account`

* EC2 security group rules

### European Central Bank (ECB)
`Public`

* Currencies
* Exchange rates

### MaxMind
`Requires free registration`

* IP locations (GeoLite2-City with English location names)
